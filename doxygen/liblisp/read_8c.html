Detailed Description

An S-Expression parser for liblisp.

Author
Richard Howe (2015)
License:
LGPL v2.1 or Later
Email:
howe..nosp@m.r.j..nosp@m.89@gm.nosp@m.ail..nosp@m.com

An S-Expression parser, it takes it's input from a generic input port that can be set up to read from a string or a file.

Todo:
Error messages could be improved. Some kind of limited evaluation could go here as well. Giving meaning to different forms of parenthesis (such as "{}", "<>", and "[]") could improve readability or be used for syntax.
Note
there is no option to strip colors of parsed input.
