void* hash_foreach ( hashtable *  h,
hash_func  func 
)

Apply "func" on each key-val pair in the hash table until the function returns non-NULL or it has been applied to all the key-value pairs. The callback might be passed NULL as either the key or the value. The hash remembers the position of the previous call to hash_foreach, a subsequent call will carry onto the next element, unless there are no more elements left in the hash.

Parameters
htable to apply func on
funcfunction to apply
Returns
void* the result of the first non-NULL function application
